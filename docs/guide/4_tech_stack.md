# 4. Technology Stack

- **Java Version:** [Java 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)+ (Tested with).
- **Framework:** [Spring Boot](https://spring.io/projects/spring-boot) 3.x.  
  Framework for the RESTful API.
- **Database:** [H2 In-Memory Database](https://www.h2database.com/html/main.html) (with [JPA](https://spring.io/projects/spring-data-jpa#overview)/[Hibernate](https://hibernate.org/orm/)).  
  For database persistence and relationship mapping and rapid demonstation.
- **Validation:** [Jakarta Bean Validation](https://beanvalidation.org/).  
- **Utilities:** [Lombok](https://projectlombok.org/) and [MapStruct](https://mapstruct.org/)  
  For reducing boilerplate and custom Mappers.
